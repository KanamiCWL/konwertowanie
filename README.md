# Konwerter Jednostek – Aplikacja Windows Forms (C#)

## Opis projektu

Prosta aplikacja okienkowa napisana w C# (Windows Forms), która pozwala na konwersję wartości między różnymi jednostkami w trzech kategoriach:

- **Długość** (metry, kilometry, mile)  
- **Waga** (kilogramy, funty)  
- **Temperatura** (Celsjusz, Fahrenheit)  

Użytkownik wybiera kategorię, jednostkę źródłową i docelową, wpisuje wartość, a aplikacja wyświetla przeliczony wynik.

---

## Funkcjonalności

- Dynamiczna zmiana jednostek na podstawie wybranej kategorii  
- Obsługa podstawowych konwersji długości, wagi i temperatury  
- Walidacja poprawności wpisanej liczby  
- Intuicyjny i prosty interfejs użytkownika  

---

## Wymagania

- System operacyjny Windows  
- .NET Framework 4.7.2 lub nowszy  
- Visual Studio (zalecane) do kompilacji i uruchomienia  
- C# w wersji 7.3 lub wyższej   

---

## Struktura projektu

- `Form1.cs` – logika aplikacji i obsługa zdarzeń  
- `Form1.Designer.cs` – definicja i układ kontrolek formularza  
- `Program.cs` – punkt startowy aplikacji  

---

## Możliwe rozszerzenia
